How CLORAG Works

Show

CLORAG is a Retrieval-Augmented Generation system that answers Cyanview support questions by combining:

  • Three knowledge sources: Docusaurus documentation, curated Gmail support threads, and custom admin documents
  • Hybrid search: Semantic understanding (Voyage AI embeddings) + keyword matching (BM25), merged via Reciprocal Rank Fusion
  • Reranking: Voyage rerank-2.5 cross-encoder re-scores top results for better relevance
  • Answer synthesis: Claude Sonnet generates coherent answers with sources and hand-drawn Excalidraw diagrams for integration scenarios

Query → Embed → Search 3 collections → Merge & Rerank → Claude generates answer
