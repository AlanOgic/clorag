Draft Auto-Reply System

Overview

The draft system automatically generates email replies for unanswered support threads using RAG search and Claude AI synthesis. It identifies pending threads, searches the knowledge base for relevant context, and creates draft replies ready for human review.

Pipeline Flow

Key Components

  • Gmail API Integration — Fetches unanswered threads and creates draft replies with proper threading headers
  • RAG-Powered Synthesis — Searches documentation and support cases to generate contextual, accurate email content
  • Pipeline Orchestration — Full process from fetching threads to creating drafts, with duplicate detection and preview support
  • Google Groups Handling — Correctly parses original sender from Google Groups headers and strips "via Group" suffixes

Camera Database

Overview

The camera compatibility database stores structured information about cameras that work with Cyanview products, including connection ports, communication protocols, and supported controls.

Data Sources

Camera data is extracted from multiple sources:

  • Documentation — Automatic extraction during docs ingestion
  • Support Cases — Extraction from resolved support threads
  • CSV Import — Bulk data management via CSV upload
  • Manual Entry — Direct additions through the admin interface

Camera Comparison

The comparison feature allows side-by-side comparison of up to 5 cameras:

  • Selection — Check boxes in the camera table to select cameras for comparison
  • Comparison View — Side-by-side display showing all specifications
  • Common Values — Shared specifications highlighted with green tags

Related Cameras

The related cameras feature finds similar cameras based on shared manufacturer, device type, ports, and protocols, with a weighted similarity scoring system.

CSV Import/Export

Bulk data management via CSV:

  • Export — Download all cameras (or filtered subset) as CSV
  • Import — Upload CSV file to create or update cameras (uses upsert logic based on name + manufacturer)

Support Cases Database

Overview

Support cases from Gmail are stored and indexed for easy retrieval, searching, and browsing. The database stores complete anonymized, analyzed content alongside raw thread data.

Full-Text Search

The database includes an FTS5 virtual table for fast full-text search with BM25 ranking across subject lines, problem/solution summaries, full document content, and keywords.

Thread Cleaning

Raw threads are cleaned before storage to remove redundant content:

  • Quoted text — Lines starting with >
  • Reply headers — "On [date], [person] wrote:" patterns
  • Forwarded headers — "---------- Original Message ----------"
  • Email signatures — "Best regards", "Sent from my iPhone", etc.

Multi-language support includes English, French, and German reply header patterns.

Answer Export

Users can export AI responses in multiple formats directly from the search results:

  • Markdown — Original formatted response with source links
  • Plain Text — Stripped formatting for pasting into emails or tickets
  • HTML — Rendered HTML for sharing or embedding
  • PDF — Downloadable document for offline use or printing

User Feedback

Thumbs up/down buttons appear on every AI answer after streaming completes:

  • Thumbs up — One-click positive feedback
  • Thumbs down — Opens optional text field for "What could be improved?"
  • One vote per search — You can change your vote at any time
Dark Mode & Animations

The admin interface supports a dark theme with automatic OS preference detection, and includes subtle animations (page transitions, card stagger, count-up stats) that respect the prefers-reduced-motion accessibility setting.
