MCP Server

Overview

The CLORAG MCP (Model Context Protocol) server exposes the full RAG knowledge base to AI assistants like Claude Desktop, Claude Code, and any MCP-compatible client. It provides tools for searching documentation, querying the camera database, browsing support cases, and accessing custom knowledge documents.

MCP Protocol

MCP is an open standard for connecting AI assistants to external data sources. The server uses FastMCP from the mcp SDK and supports both stdio (local) and streamable-http (remote) transports.

Search Tools

Hybrid RAG search across all three vector collections with RRF fusion and Voyage reranking.

Tool async search
Search all sources using hybrid RAG with reranking. Main entry point for queries.
query: str · source: "docs" | "cases" | "custom" | "both" · limit: 1-20

Camera Tools

Camera compatibility database with FTS5 search, comparison, and similarity.

Tool sync search_cameras
Full-text search across camera names, manufacturers, model codes, ports, protocols, and notes.
query: str (e.g. "Sony PTZ", "VISCA", "SDI")
Tool sync get_camera
Get full details of a camera by ID.
camera_id: int
Tool sync list_cameras
List cameras with optional filtering by manufacturer, device type, or protocol.
manufacturer: str? · device_type: str? · protocol: str? · limit: 1-50
Tool sync find_related_cameras
Find similar cameras (same manufacturer, type, ports, or protocols).
camera_id: int · limit: 1-10
Tool sync compare_cameras
Side-by-side comparison of 2-5 cameras with common ports and protocols highlighted.
camera_ids: list[int] (2-5)
Tool sync get_camera_stats
Database statistics: total cameras, manufacturers, device types, ports, protocols.

Document Tools

Custom admin-managed documents across 10 categories.

Tool async list_documents
List custom documents with optional category filter and pagination.
category: str? · limit: 1-50 · offset: int
Tool async get_document
Get full content and metadata of a custom document.
doc_id: str (UUID)

Support Case Tools

Curated Gmail support cases with FTS5 search and raw thread access.

Tool sync search_support_cases
Full-text search across subject, problem/solution summaries, document content, and keywords.
query: str · limit: 1-20
Tool sync get_support_case
Get full case details with problem/solution summaries and document content.
case_id: str
Tool sync get_support_case_raw
Get the raw anonymized email thread (not just summaries). Full conversation flow with quoted replies.
case_id: str
Tool sync list_support_cases
List cases with optional category and product filters.
category: str? · product: str? · limit: 1-50 · offset: int
Tool sync get_support_stats
Statistics: total cases, categories, products, resolution quality breakdown.

Resources vs Tools

Resources are for reading data (clients can subscribe to updates). Tools are for actions (search, edit, ingest). Use resource templates when you know the exact ID; use tools when you need to search or filter.

Prompt Templates

Pre-built conversation starters that guide the AI assistant through structured workflows.

Prompt troubleshoot
Guided troubleshooting: searches docs + cases, identifies causes, provides step-by-step resolution.
issue: str (required) · product: str (optional)
Prompt compare_cameras
Side-by-side camera comparison for Cyanview compatibility. Searches DB, highlights ports/protocols.
camera_names: str (required)
Prompt find_camera
Find cameras matching a use case. Recommends Cyanview products and connection methods.
use_case: str (required) · requirements: str (optional)
Prompt explain_product
Detailed product explanation: variants, features, use cases, licensing, connections.
product: str (required, e.g. "RIO", "CI0")
Prompt integration_guide
Generate a full integration guide: hardware, network, step-by-step setup, pitfalls.
scenario: str (required)
Prompt ingest_status
Analyze knowledge base health and suggest which ingestion operations to run.
